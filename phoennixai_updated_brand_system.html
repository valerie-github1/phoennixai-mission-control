01 — Brand Decision
Why We Are Keeping the Blue Phoenix
Customer Feedback Is Real Data

Valerie's feedback from customers: The fire and ice direction caused confusion. People did not understand why a blue phoenix logo was being paired with orange and gold fire messaging.

Valerie's personal preference: Blue is her favorite color. The blue phoenix feels right. It always has.

Strategic decision: We are keeping the blue phoenix as the primary logo and making Phoenix Blue (#00E5FF) the primary brand color. The fire colors (Phoenix Fire #FF6B2B and Ember Gold #FFB830) are repositioned as accent colors and agent colors, not core brand colors.

What this means:

Phoenix Blue is now the primary brand color. It represents intelligence, clarity, trust, and the electric energy of the PhoennixAI mission. This color appears in the logo, the primary wordmark, all primary brand touchpoints, and as ARIA's agent color.

Sapphire Wing (#46ABD7) becomes the secondary brand color. It appears in gradient backgrounds, supporting UI elements, and as a softer blue alternative where Phoenix Blue would be too intense.

Phoenix Fire (#FF6B2B) and Ember Gold (#FFB830) are repositioned as accent colors. They are used for calls to action, highlights, and specific agent identities (Claude is Fire, LEDGER is Gold), but they are no longer the primary brand colors.

This creates a cleaner, more cohesive brand system that matches the logo customers already know and love.

Light Background Specifications:

  • AI: Charcoal #111827 (dark grey/black for contrast)

Use the wordmark-only version (without the phoenix bird) in these situations: app headers where space is limited, mobile navigation bars, email footers, favicon alternatives, and anywhere the full logo would be too large or detailed to render clearly. The wordmark colors always match the logo: cyan "Phoennix" and grey "AI".

Why agent colors matter:

Each agent has their own color identity. This allows users to instantly recognize which agent is speaking, which agent owns which feature, and which agent is responsible for which part of their workflow.

When ARIA sends a morning briefing, it appears in Phoenix Blue. When CLAUDE writes a strategy document, the header is Phoenix Fire. When SCRIBE saves a version, the notification is Bronze. This creates a visual language that makes the AI feel like a team, not a tool.

06 — Usage Guidelines
How to Apply the Brand System

Primary Applications (Use Phoenix Blue #00E5FF):

  • Logo (the phoenix bird is blue)
  • Primary CTAs (buttons, links)
  • Navigation highlights
  • Section headers on website
  • ARIA agent identity
  • Social media profile accents

Secondary Applications (Use Sapphire Wing #46ABD7):

  • Gradient backgrounds (from Phoenix Blue to Sapphire)
  • Supporting UI elements
  • Softer accents where Phoenix Blue is too intense
  • Hover states

Accent Applications (Use Phoenix Fire #FF6B2B and Ember Gold #FFB830):

  • Highlights and emphasis
  • Warning states (Fire)
  • Success states (Gold)
  • Agent-specific branding (Claude is Fire, LEDGER is Gold)
  • Special announcements or promotions

Typography (Locked System):

  • Display: Bebas Neue (all headlines, wordmarks, agent names)
  • Body: DM Sans (300, 400, 500, 600) — all UI text and body copy
  • Accent: Playfair Display (italic) — special emphasis only
  • Code/Labels: DM Mono — technical text, labels, metadata
Brand Rule: No Em Dashes Ever

This rule stays locked. No em dashes (—) anywhere in any PhoennixAI output. Use commas, full stops, or restructure. This applies to all agents, all marketing copy, all documentation, and all customer-facing content.

What Changed and What Stayed the Same

Changed: Phoenix Blue (#00E5FF) is now the primary brand color instead of Phoenix Fire (#FF6B2B). The blue phoenix logo is the hero logo for all primary applications. Fire and Gold are now accent colors, not core brand colors.

Stayed the same: Typography system (Bebas Neue, DM Sans, etc.). Agent color identities. Tagline "Rise. Automate. Dominate." Brand voice and messaging. No em dashes rule. Midnight background (#060A18) for dark mode.

Result: A cleaner, more cohesive brand system that matches the logo customers already know and love. Less confusion. More trust. Better alignment between visual identity and brand promise.
